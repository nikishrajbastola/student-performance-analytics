# Student Performance Dashboard

## Overview
Interactive Streamlit dashboard analyzing student performance.

## Features
- Study time vs grades
- Absences vs performance
- Alcohol consumption vs performance
- Filters for study time and failures

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Key Insights
- More study time → better grades
- Past failures → lower performance
- Absences negatively impact grades
